# Product_Management

**Product_Management** is a PHP-based web application for managing products. It features a full **CRUD system** (Create, Read, Update, Delete) and a modern **AdminLTE dashboard** for easy product management.

---

## Features
- **Create, Read, Update, Delete** products
- View all products in the **AdminLTE dashboard**
- Database integration using MySQL
